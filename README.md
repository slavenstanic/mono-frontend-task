# Vehicle App (Mono Frontend Task)

# Local Development

## Prerequisites:
- Install `node` (tested on v20.17.0)
- Install `pnpm` (tested on v10.4.1)



## Getting started:
1. Navigate to `/mono-frontend-task`
2. Install dependencies:
   ```bash
   pnpm install
3. Create a `.env` file based on `.env.example`
4. Run `pnpm run dev`
